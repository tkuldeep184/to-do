To-Do List Application

This To-Do List app is a web-based application built using HTML, CSS, JavaScript, and Express.js. It allows users to manage their daily tasks efficiently by adding, deleting, and prioritizing tasks. The app is designed to be user-friendly and responsive, making it accessible from both desktop and mobile devices.

Key Features:

1. Add New Tasks: Users can quickly add new tasks to their to-do list using a simple input form.

2. Delete Tasks: Completed or unnecessary tasks can be easily removed from the list.

3. Priority Management: Users can increase or decrease the priority of tasks, helping them focus on the most important items.

4. Persistent Data: The app ensures that your tasks are not lost upon refreshing the page, providing a seamless experience even if the browser is closed or refreshed.

This To-Do List app is perfect for anyone looking to organize their tasks and manage their time more effectively
